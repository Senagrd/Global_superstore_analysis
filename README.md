# Global_superstore_analysis
Bu proje, 2011–2014 yılları arasındaki online süpermarket siparişlerini analiz ederek e-ticaret müşteri davranışlarını ve kârlılık yapısını incelemektedir. Veri seti eksiksiz ve temel analizler için uygundur. Amaç, sadece grafik üretmek değil; müşteri davranışlarını anlamak ve veri odaklı iş kararları geliştirmektir.
